Layer-by-Layer Spec Explorer

Explore each of the 12 layers with intra-layer and inter-layer relationship views


This guide provides two query patterns for each layer:

  • Intra-layer view - Nodes and relationships within a single layer

  • Inter-layer view - Nodes and relationships crossing layer boundaries

Each layer gets two slides:

  • Slide A: Internal structure (intra-layer relationships)

  • Slide B: Cross-layer connections (inter-layer relationships)

Tip: Use the arrow buttons at the bottom to navigate between layers!

Layer 1: Motivation (Intra-layer)

Internal relationships within the Motivation layer


All Motivation nodes with intra-layer relationships:

// Find all Motivation layer nodes
MATCH (layer:Layer {id: 'motivation'})
MATCH (node:SpecNode)-[:BELONGS_TO_LAYER]->(layer)

// Find intra-layer relationships
OPTIONAL MATCH (sr:SpecRelationship)-[:HAS_SOURCE]->(node)
OPTIONAL MATCH (sr)-[:HAS_TARGET]->(target:SpecNode)-[:BELONGS_TO_LAYER]->(layer)
OPTIONAL MATCH (sr)-[:HAS_PREDICATE]->(pred:Predicate)

RETURN node, sr, target, pred, layer
LIMIT 100;

Shows: Goals, stakeholders, drivers, assessments, constraints, requirements, principles

Layer 1: Motivation (Inter-layer)

Cross-layer relationships for Motivation nodes


Motivation nodes with inter-layer relationships:

// Find Motivation layer nodes
MATCH (layer1:Layer {id: 'motivation'})
MATCH (node:SpecNode)-[:BELONGS_TO_LAYER]->(layer1)

// Find inter-layer relationships (both directions)
OPTIONAL MATCH (sr:SpecRelationship)-[:HAS_SOURCE]->(node)
OPTIONAL MATCH (sr)-[:HAS_TARGET]->(target:SpecNode)-[:BELONGS_TO_LAYER]->(layer2:Layer)
WHERE layer1 <> layer2
OPTIONAL MATCH (sr)-[:HAS_PREDICATE]->(pred:Predicate)

RETURN node, sr, target, pred, layer1, layer2
LIMIT 100;

Shows: How Motivation influences Business, Security, and other layers

Query Tips

Making the most of layer exploration


Adjust the LIMIT:

All queries use LIMIT 100 to prevent overwhelming graphs. Increase or decrease based on your needs:

// Show more results
LIMIT 200

// Show fewer results for cleaner graphs
LIMIT 25

Filter by specific predicates:

Add a WHERE clause to focus on specific relationship types:

WHERE pred.id = 'composes'
// or
WHERE pred.id IN ['realizes', 'influences', 'serves']

Focus on specific node types:

Filter nodes by their type:

WHERE node.id STARTS WITH 'motivation.goal'
// or
WHERE target.id CONTAINS 'endpoint'

Use the graph browser filters:

  • Click the colored circles in the right sidebar to show/hide node types

  • Use the search box to find specific nodes

  • Double-click nodes to expand their relationships
